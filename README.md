# P-rolaNoivas
📝 README.md
👰 Projeto Landing Page - Pérola Noivas
Este projeto consiste no desenvolvimento de uma Landing Page sofisticada para a loja Pérola Noivas, localizada em Brasília. O objetivo foi transformar a presença visual do Instagram da loja em uma interface web funcional, focada em experiência do usuário (UX) e conversão de vendas.

🎨 Sobre o Projeto
A página foi construída com um Boxed Layout (layout em caixa) para transmitir exclusividade e organização, utilizando conceitos avançados de Flexbox para garantir que o conteúdo seja centralizado e responsivo.

Identidade Visual: Baseada em tons de Off-white, Dourado Champanhe e Grafite de alto contraste.

Destaques: Seções dedicadas a grifes internacionais (Pollardi, Rosa Clará, Pronovias), curadoria de acessórios e o setor de Bazar de vendas.

Objetivo: Facilitar o agendamento de horários e exibir a autoridade de 21 anos de tradição da marca.

🛠️ Tecnologias Utilizadas
HTML5: Estrutura semântica para melhor SEO.

CSS3: Estilização personalizada com variáveis e Flexbox.

Google Fonts: Utilização das fontes Playfair Display (Serifada/Luxo) e Montserrat (Sans-serif/Moderna).
